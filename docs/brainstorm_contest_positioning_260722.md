## 1. 대회가 보는 것 (요약)

- **국내 LLM(Solar) 메인** 활용이 드러날 것  
- 실생활·개인 적용 (부모님 보호 = 강 축)  
- 자기주도 시행착오 (프롬프트 QA, 배포, UX 반복)  
- 윤리 (저장 안 함, 출처, AI 한계)  
- 표현력 (이해하기 쉬운 구성)  
- **기능 폭주 < 작은 완성 + 과정 스토리**

## 2. 제품 한 줄 (현재 가설)

**가설 피치 (심사위원용, Claude 정리 기반):**

> 안심체크는 부모 폰에 붙여 두는 문자 위험 검사기이며, Upstage Solar가 위험 점수·이유·할 일을 말하고, **광고가 붙을 자리에는 경찰청 안전Dream 공개 정보를 올리는 공익 슬롯**을 둔 생활형 AI입니다.

---

## 3. 실종 보드 — 왜 명분이 약해 보였는지

1. 메인 문제(의심 연락)와 행동(얼굴 기억)이 **다른 순간**  
2. Solar 서사와 OpenAPI 서사가 **끊겨 보임**  
3. “좋은 기능인데 왜 여기?” → 심사 시 **산만** 리스크  
4. 제작 의도(광고 대체 슬롯)가 **화면에 안 쓰여 있으면** 과시/부가기능으로 읽힘  

→ **유지 자체는 타당**. 문제는 기능이 아니라 **읽히는 문장**.

## 4. 실종 보드 프레이밍 후보 (가설)

### 추천 조합 (Claude)

| 층 | 전략 | 상태 |
|----|------|------|
| **화면 UI** | “안전 소식 / 공익 자리” 톤 (과하지 않게) | 가설 |
| **README·발표** | “광고 자리 대신 공익 할당” 명시 | 가설 |
| **금지** | 이용자를 수사 주체로 만드는 말, Solar가 실종 분석한다는 혼동 | 가설 |

### 명분 문장 후보 (가설 — 카피 확정 아님)

**A. 공익 자리 (광고 슬롯 은유, 가장 의도에 가깝)**  
- *여기는 광고가 붙지 않는 자리입니다. 그 자리에 공공 안전을 둡니다.*  
- *광고 대신, 경찰청 공개 정보를 보여 드립니다.*  
- *이 앱은 광고로 돈을 벌지 않습니다. 대신 이 칸에 OpenAPI 공익 정보를 올립니다.* (발표용 — 화면엔 짧게)

**B. 다급함 이후 한 호흡 (플로우 통합)**  
- *급한 문자를 보셨다면, 확인 뒤에 잠깐 숨을 고르세요.*  
- *그 사이, 우리 주변에서 찾는 얼굴을 한 번 기억해 주세요.*

**C. 고립 깨기**  
- *혼자가 아닐 때 사기에 덜 속습니다. 판단(문자)과 연결(공공 정보)을 한 화면에.*

**Grok 추가 후보**  
- *사기 문자는 사람을 급하게 만든다 → 확인 후 공익 슬롯으로 “급함”을 끊는다.*  
- *Solar로 판단하고, 공공 API로 사회에 한 칸 내놓는다.* (기술 스택 이중 자랑)

### 카피 톤 규칙 (가설)

**해도 됨**  
- 출처: 경찰청 안전Dream  
- 의도 1문장: 광고 자리 대신 공익  
- 행동 1개: 공식 사이트 / 182  
- 담담한 은행 앱 하단 공익 배너 톤  

**하지 말 것**  
- “함께 수사해요”, 공포·선정 과장  
- AI가 실종을 분석한다는 오해  
- 내부 용어(OpenAPI flex, 슬롯, draft)를 사용자 화면에 그대로  
- 광고 비하 과함 (“구글 광고 따위”)

### 비판 한 줄 (Claude, 채택 가치 있음)

> **OpenAPI 플렉스를 화면이 말하면 지고, “광고 자리의 공익 할당”을 화면이 말하면 이긴다.**  
> (기술 자랑은 README·발표, 화면은 담담한 공익 배너.)

|------|------|-----|-------------|

## 7. 멀티 AI 교차 메모

### Claude (subagent, 2026-07-22)

- UI = 안전 소식/담담 배너, 스토리 = 광고 대신 공익  
- 과자 뒷면 비유는 대외 카피에서 약함 → 동작만 유지 가능  
- 실종 유지 가능하나 **프레이밍 없으면 리스크** (초점 분산, 윤리 오해, Solar 희석)  
- 한 문장 피치 제공 (위 §2)

### NVIDIA NIM / DeepSeek V4 Flash (2026-07-22)

- 호출 성공했으나 **Solar(LLM)를 태양광·배터리로 오해**하는 응답이 섞임 → **품질 낮음, 참고만**  
- 유효 포인트 1: OpenAPI 지연/오류 시 **면책·검증 고지** 필요 (윤리)  
- 피치·PWA 아이디어 중 상당수는 **제품 맥락 이탈** → 채택 보류

### Grok (본 세션)

- 실종 = 광고 슬롯 대체라는 사용자 의도를 **의도적 제품 윤리 선언**으로 승격하는 방향 지지  
- 화면: 티 내되 과시 톤 금지 / README: OpenAPI·무광고 명시  
- 다음 구현 후보: ① 실종 구역 카피 보강 ② PWA 홈 추가 ③ 제출용 스토리 문단
